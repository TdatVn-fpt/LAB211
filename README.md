# LAB211 - Object-Oriented Programming Lab 🚀

Chào mừng đến với repository lưu trữ tài liệu và mã nguồn thực hành cho môn học **LAB211** tại Đại học FPT. Repository này được sử dụng để quản lý tiến độ học tập, lưu trữ đề bài, tài liệu ôn tập và mã nguồn của các bài Lab.

## 📂 Cấu trúc Repository

Repository được tổ chức thành các thư mục sau để dễ dàng tra cứu:

* **`lab211_debai/`**: Chứa các file tài liệu mô tả yêu cầu đề bài của các bài Lab thực hành.
* **`Passlab_hienDa/`**: Nơi lưu trữ mã nguồn (Source Code) của các bài Lab đã hoàn thiện và code pass.
* **`Silde_inclass/LAB211-REVIEW/`**: Bao gồm các slide bài giảng trên lớp và tài liệu tổng hợp kiến thức (Review) để chuẩn bị cho các bài kiểm tra thực hành.
